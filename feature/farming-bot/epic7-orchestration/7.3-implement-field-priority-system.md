## Technical Notes
- Priority calculation example:
  ```
  if (harvestable_crops > 0 && time_since_harvestable > threshold):
      priority = CRITICAL
  elif (harvestable_crops > 0):
      priority = HIGH
  elif (tilled_tiles > 0 && seeds_available):
      priority = MEDIUM
  elif (growing_crops > 0):
      priority = LOW
  else:
      priority = IDLE
  ```
